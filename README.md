# Emotion Recognition & IoT Adaptive System

## 📌 Overview
The Emotion Recognition & IoT Adaptive System is an intelligent web-based application that detects human emotions using facial expressions and adapts system responses accordingly. It integrates AI-based emotion detection with IoT concepts to provide a smart, responsive environment.

---

## 🚀 Features
- 🎭 Real-time emotion detection using camera
- 🧠 AI-based facial expression analysis
- 🌐 Web-based interactive UI (Login & Dashboard)
- 📊 Emotion history tracking and storage
- 🔊 Voice + facial emotion integration (future scope)
- ⚡ Adaptive response system based on user emotion
- 📱 Responsive design for mobile and desktop

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Python (Flask)  
- Database: SQLite  
- Libraries: OpenCV, NumPy, etc.  
- Visualization: Chart.js / D3.js

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
git clone https://github.com/your-username/emotion-recognition-and-iot-adaptive-system.git

### 2️⃣ Navigate to project folder
cd Emotion-Recognition-System

### 3️⃣ Create virtual environment
python -m venv venv

### 4️⃣ Activate environment
venv\Scripts\activate   (Windows)

### 5️⃣ Install dependencies
pip install -r requirements.txt

### 6️⃣ Run the application
python app.py
